# AI Auto Email Reply using n8n

## Description
This project automatically reads incoming emails and generates AI-based replies using n8n and OpenAI.

## Workflow
Gmail Trigger 
        ↓
Read Email Content
        ↓
AI Model (OpenAI)
        ↓
Generate Reply
        ↓
Send Email Automatically

## Features
- Automatically detects new emails
- Understands email content using AI
- Generates professional replies
- Sends response automatically

Technologies Used
- n8n
- Gmail API
- OpenAI API

## Setup
1. Import workflow.json into n8n
2. Connect Gmail credentials
3. Add OpenAI API key
4. Activate workflow
